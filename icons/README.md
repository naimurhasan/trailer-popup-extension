# Icons

Please add the following icon files to this directory:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can create these icons using any image editor. They should represent a movie/video theme (e.g., a play button, film reel, or popcorn).

Recommended tools for creating icons:
- Figma (free, online)
- Canva (free templates available)
- GIMP (free, desktop)
- Adobe Photoshop (paid)

Or use online icon generators like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
